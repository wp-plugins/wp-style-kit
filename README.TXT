== Description ==

WP Style Kit allows you to easily apply styles to text in the Wordpress Visual Editor from a number of predefined styles.

WP Style Kit is an easy–to–use WordPress plugin that gives you more styles for your pages or posts.

It's fast to install and requires zero technical background. You can apply beautiful custom styles, such as additional headlines, custom bullets, quotes, and callout boxes.

Learn how to install and use the WP Style Kit Wordpress Plugin, which allows you to apply custom styles to your text, images, and videos to your WordPress blog posts and pages:
[youtube https://www.youtube.com/watch?v=16MhImCjfLk]

== Installation ==

1. Install WP Style Kit either via the WordPress.org plugin directory, or by uploading the entire wp-style-kit folder to the /wp-content/plugins/ directory on your server.

2. Activate the plugin through the ‘Plugins’ menu in WordPress.

3. From the “Formats” menu of the Visual Editor, select the style that you want to apply.

== Frequently Asked Questions ==

= I can’t see the Formats menu =

In the Visual Editor, click “Toolbar Toggle” to display the second row of WordPress menus icons. The “Formats” menu is on the left of the “Styles” menu. 

= Can I add a border to images or videos? =

Yes, you can use the Shadow Box style.
